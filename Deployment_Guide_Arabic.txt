دليل رفع مشروع Smart Leaf Doctor على الإنترنت مجاناً 🚀
======================================================

أولاً وقبل أي شيء: الصورة اللي إنت صورتها للتطبيق (اللي مش عاجباك) دي كانت من أداة Streamlit المؤقتة!
أنا بالفعل حققت لك حلمك وصممت الواجهة الاحترافية اللي من Stitch بالمللي وبشكل ديناميكي في ملف اسمه `index.html` جوه مجلد اسمه `frontend` في مشروعك. 
لو فتحت الملف ده دلوقتي مرتين بالماوس، هتلاقيه هو هو نفس تصميم Stitch بالضبط وشغال مع الذكاء الاصطناعي!

ثانياً: خطة الرفع على الإنترنت (Hosting)
بما إنك قررت ترفع المشروع على النت عشان تاخد الـ 25/25 وإنت حاطط رجل على رجل، فالمشروع بتاعنا دلوقتي عبارة عن جزئين:
1. واجهة أمامية (Frontend): اللي هي ملف الـ HTML الاحترافي.
2. خادم خلفي (Backend): اللي هو ملف الـ Flask اللي بيشغل الذكاء الاصطناعي والموديل.

أفضل وأقوى منصة مجانية في العالم لرفع مشاريع الذكاء الاصطناعي (البايثون) حالياً هي: Render.com

خطوات الرفع بالتفصيل الممل:
--------------------------
الخطوة 1: تجهيز المشروع للرفع
1. تأكد إن ملف `smart_leaf_doctor_mobilenetv2.h5` موجود جوه مجلد `models`.
2. في مجلد المشروع الرئيسي، تأكد من وجود ملف `requirements.txt` وفيه المكتبات دي:
   tensorflow
   Flask
   flask-cors
   Pillow
   gunicorn

الخطوة 2: رفع الكود على GitHub (مخزن الأكواد)
1. ادخل على موقع https://github.com واعمل حساب مجاني.
2. اضغط على علامة (+) فوق واختار "New Repository" (مشروع جديد).
3. سَمّي المشروع "Smart-Leaf-Doctor" واجعله Public، ثم اضغط "Create repository".
4. ارفع كل ملفات مجلد المشروع بتاعك (ما عدا مجلد الداتا سيت اللي مساحته 2 جيجا عشان مش محتاجينه على النت، إحنا محتاجين الموديل الـ 17 ميجا بس).

الخطوة 3: ربط GitHub بمنصة Render (للتشغيل الفعلي)
1. ادخل على موقع https://render.com واعمل حساب مجاني باستخدام حساب GitHub بتاعك.
2. من لوحة التحكم، اضغط على "New" واختار "Web Service".
3. اختار "Build and deploy from a Git repository".
4. هتلاقيه جابلك اسم المشروع بتاعك من GitHub، اضغط "Connect".
5. في الإعدادات اللي هتظهر:
   - Name: اكتب اسم المشروع (مثلاً: smart-leaf-doctor)
   - Runtime: اختار Python 3
   - Build Command: اكتب `pip install -r requirements.txt`
   - Start Command: اكتب `gunicorn backend.server:app`
6. انزل تحت واختار الخطة المجانية "Free".
7. اضغط على زرار "Create Web Service".

الخطوة 4: مبروك!
المنصة هتاخد حوالي 5 إلى 10 دقائق تحمل المكتبات وتشغل الذكاء الاصطناعي.
بعدها هتديلك لينك (مثلاً: https://smart-leaf-doctor.onrender.com).
هذا اللينك هو اللي هتربطه بملف الـ HTML بتاعك، وتاخد ملف الـ HTML ده وتفتحه من أي مكان في العالم أو ترفعه على GitHub Pages، وبكده تكون عملت سيستم كامل متكامل!

ملاحظة هامة: أنا جاهز أمشي معاك خطوة بخطوة في الرفع لو احتجت أي مساعدة أثناء التنفيذ.
